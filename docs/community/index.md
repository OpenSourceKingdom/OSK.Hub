# Community

Welcome to the OSK community. We value collaboration, transparency, and respectful discourse.

## Governance

All contributors and members are expected to follow our **[Code of Conduct](https://github.com/opensourcekingdom/.github/blob/main/code_of_conduct.md)** and related governance files. These documents outline the standards for participation and conflict resolution within the organization.

## Feedback, Bugs, Feature Requests

Any feedback, criticism, bug reports, or feature requests are appreciated. To help build better software for all, community engagement is required. Please utilize the repository issue tab to create bugs or feature requests and, if for some reason your feedback doesn't quite fit the repository, you can use the contact page for communication.

## Contributing

We welcome contributions to all OSK projects. To contribute:

1. Fork the repository being updated (or the specific project repo)
2. Create a branch (`git checkout -b <branch-name>`)
3. Make your changes and push the branch to your fork
4. Submit a pull request from your fork to this repository

**By submitting a pull request, you affirm that:**

- You have the right to submit the contribution under the project's license
- Your contribution does not infringe on any third-party intellectual property rights
- You agree to abide by the organization's Code of Conduct

This lightweight Contributor Agreement ensures all contributions are voluntary and properly licensed. It applies to every PR submitted to any OSK repository.
