There are a vast number of projects available in OSK. Below are some of the notable projects that can be discovered:

## DeepEquals

A fast and flexible extension that supports deeply comparing objects. See [project repository](https://github.com/OpenSourceKingdom/OSK.Extensions.Object.DeepEquals). 

## Outputs

Provides an implementation of the result pattern and is designed to work with paging. See [project repository](https://github.com/opensourcekingdom/OSK.Operations.Outputs).

## Messages

A system that transmits, receives, and distributes messages between and in applications. Supports a variety of message couriers, for either local ([example](https://github.com/OpenSourceKingdom/OSK.Messages.Couriers.Pigeons)) or distributed ([example](https://github.com/OpenSourceKingdom/OSK.Messages.Couriers.EasyNetQ)) messaging.

## Cryptography

Easy to use access to cryptographic services via standard .NET DI and cryptographic key parameters. Made to support symmetric and asymmetric keys. See [project repository](https://github.com/OpenSourceKingdom/OSK.Security.Cryptography).

## Storage Service

Data persistence and retrieval services that can be easily integrated into .NET applications. Designed to provide a consistent API service that can be used across distributed or local ([example](https://github.com/OpenSourceKingdom/OSK.Storage.Local)) storage.

## UML Diagrams

C# UML diagram generation direct from code. See the [project repository](https://github.com/OpenSourceKingdom/OSK.UML).

## Invoker

Highly performant runtime execution of functions on generic objects. See [project repository](https://github.com/OpenSourceKingdom/OSK.Expressions.Invoker).
